Repository contents:

stata_do_file - analysis code
aurum_final - Diabetes codes used to identify diabetes cohort in CPRD Aurum
GOLD_final - Diabetes codes used to identify diabetes cohort in CPRD GOLD

Note on codes: in both aurum and GOLD, any generic diabetes codes (type=0) and type 2 diabetes codes (type=2) were used to identify individuals with type 2 diabetes, whereas individuals with any type 1 diabetes code (type =1) were excluded.
